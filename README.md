# DA40-D Mass & Balance and Performance Calculator

Pre-flight Mass & Balance and performance calculation tool for the Diamond DA40-D, operated by NewCAG fleet.
Base on the AFM Doc. #6.01.05-E Rev.7 · 27-Sep-2013

## Features

- Mass & Balance
- Take-Off Distance Required (TODR)
- Landing Distance Required (LDR)
- Landing Roll Required (LRR)
- PDF export

## Disclaimer

For planning purposes only. Not for operational use.
Verify all figures against the DA40-D AFM — authoritative source at all times.
